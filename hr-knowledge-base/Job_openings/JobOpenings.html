Job Description

This section is used by the admin to manage andmonitor all job postings. Each entry in the table below

File Description
Code Unique job post identifier for tracking purposes
Title The role or position offered (e.g Java Developer, Front endDeveloper).
Location Indicates the branch or area where the job is based.
Open Posts Number of vacancies available for the role.
Created at The date of which the job was posted
Status Reflects the current state of the Job post (Active, Draft, Initial Screening).
Experience Years of Experience want for the job to Eligible
Action Options to edit or delete the respective job entry.

Use the search Box to filter Jobs. The dropdown allows viewing more than 10 entries at once.
